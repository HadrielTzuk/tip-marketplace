Siemplify Marketplace Release Notes - 14.6

Published on 10 March, 2021

Integrations

  • CSV
    • CSV
      • Added new optional parameters for actions "Search by Entity" and "Search by String"
      • Updated output messages for actions "Search by Entity" and "Search by String"
      • Added ability to provide multiple columns in the parameter "CSV Column" for actions "Search by Entity" and "Search by String"
      • Updated action and action parameter descriptions for actions "Search by Entity" and "Search by String"
      • Added ability to apply custom "Alert Name" and "Severity" for alerts ingested by "CSV Connector"
      • Updated connector and connector parameter descriptions in the "CSV Connector"
  • Exchange
    • Exchange
      • Integration configuration is updated to support Oauth authentication.
  • ServiceNow
    • ServiceNow
      • Added support for Oauth authentication. Please refer to the integration documentation for all of the required steps.
