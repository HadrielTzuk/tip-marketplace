Siemplify Marketplace Release Notes - 17.05

Published on 16 February, 2022

Integrations

  • Exchange
    • Exchange
      • Mail Connector v2 with Oauth Authentication, Mail Connector v2 - Added an ability to provide custom alert name templates.
      • Exchange EML Connector, Exchange Mail Connector v2, Exchange Mail Connector v2 with Oauth Authentication - Updated the logic behind "Mark Emails As Read" parameter - Connector test runs will no longer mark the emails as "Read".
  • Tanium
    • Tanium
      • Updated the integration to work with Tanium Platform REST APIs, with API token based authentication. Note - Version 5 of the integration is not operational, please make sure to upgrade the integration to version 6 and make sure your workflows are checked and updated in the case of need - JSON results can be different.

Actions

  • Exchange
    • Download Attachments
      • REGRESSIVE - Download Attachments action - added handling of slash (/) and backwardslash (\) characters in the email subject, so downloaded attachments that have "/" or "\" characters in the names, will have those characters replaced with the "_" character.
    • Move Mail To Folder
      • REGRESSIVE! REGRESSIVE - Move Mail to Folder action - By default action limits the amount of data returned in the action's json result. Make sure your playbooks are not relying on the json result the action returned before. If so, please update the new additional parameters according to your needs. Please refer to the integration documentation for more information.
  • VirusTotalV3
    • Enrich URL
      • Enrich URL - Added fallback value for the parameter "Resubmit After (Days)"

Connectors

  • MicrosoftDefenderATP
    • Microsoft Defender ATP Connector V2
      • Microsoft Defender ATP Connector v2 - Added mapping for the default configuration of the connector.
