Siemplify Marketplace Release Notes - 12.11

Published on 26 May 2019

Actions

  • Jira
    • Integration authentication changed from basic authentication to token

  • Zscaler - New Integration
    • Add to Blacklist
    • Add to Whitelist
    • Remove from Blacklist
    • Remove from Whitelist
    • Get Blacklist
    • Get Whitelist
    • Get sandbox report
    • Get URL categories
    • Lookup entity
    • Ping

  • Anomali
    • Integration was aligned to support API changes
    • Get Threat Info
      • Entity lookup was changed to match the exact string
      • Improved suspicious entities logic
      • Output message changed to be more informative

  • Siemplify
    • Modified description for the following actions:
      • Add to custom list
      • Is in custom list
    • Create entity outcome modified to be more informative

  • CA Service Desk
    • Add support for Unicode in all actions

  • Active Directory
    • Enrich Entities
      • Script result was modified to is success (in addition to JSON Result)

  • IntSights
    • Added support for new API

  • Publisher integration was removed from marketplace
    • Siemplify 5.0 architecture no longer using this integration

    Connectors

  • Mcafee ESM Connector
    • Improved environment logic
      • Added 'Use Mcafee Environments' checkbox - ingest cases to Siemplify environment by Mcafee multi-tenant mechanism
      • Added map.json file to support use case of unequal environment names in Mcafee and Siemplify
    • Added option to exclude alerts from ingesting into Siemplify using blacklist

  • QRadar New Correlations Events Connector
    • Improved environment logic
      • Added 'Use Qradar Environments' checkbox - ingest cases to Siemplify environment by Qradar multi-tenant mechanism
      • Added map.json file to support use case of unequal environment names in Qradar and Siemplify
