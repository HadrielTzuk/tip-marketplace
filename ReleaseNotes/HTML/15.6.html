Siemplify Marketplace Release Notes - 15.6

Published on 27 July, 2021

Integrations

  • CBCloud
    • VMware Carbon Black Cloud
      • VMware Carbon Black Cloud Alerts and Events Baseline Connector, VMware Carbon Black Cloud Alerts and Events Tracking Connector - Additional fields added to Siemplify alerts, created from the Carbon Black Cloud alerts.
  • FireEyeHX
    • FireEye HX
      • Added support for multiple Alert Group IDs in actions "Get Alert Group Details" and "Get Alerts in Alert Groups".
  • Splash
    • Splash
      • NEW! New integration "Splash".
  • TenableIO
    • Tenable.io
      • NEW! New integration "Tenable.io"

Actions

  • CiscoThreatGrid
    • Upload Sample
      • Added an ability to submit files from remote servers in action "Upload Sample".
  • FireEyeHX
    • Get Alerts
      • Added information about Alert Group ID in action "Get Alerts".
    • Get Host Alert Groups
      • NEW! Added new action "Get Host Alert Groups"
  • Zendesk
    • Create Ticket
      • Added an ability to provide "Email CCs" in action "Create Ticket".

Connectors

  • EmailV2
    • Generic IMAP Email Connector
      • Generic IMAP Email Connector - New parameters "Original Received Mail Prefix" and "Attached Mail File Prefix" added, in order to prepend a prefix to the keys (to,from,subject...) extracted from the original email received in the monitored mailbox, and attached mail files respectively.
      • Generic IMAP Email Connector - Added the ability to create multiple Siemplify alerts in a case of a mail containing multiple attachments - 1 alert per attached mail file.
      • Generic IMAP Email Connector - Added support for different event types, in case of ingesting an original EML received in the monitored mailbox, and when ingesting as an attached EML. Please refer to our documentation for additional information
      • Generic IMAP Email Connector - Updated handling of the email files, in cases of attachments that are missing required headers.
      • Generic IMAP Email Connector - Added support for Emails that contain no direct recipients, but contain CC or BCC recipients.
