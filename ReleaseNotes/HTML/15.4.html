Siemplify Marketplace Release Notes - 15.4

Published on 30 June, 2021

Integrations

  • CBCloud
    • VMware Carbon Black Cloud
      • Updated the integration's code to work with Python version 3.New "Execute Entity Processes Search" action added
      • Improved functionality, error handling and outputs throughout the integration's components
      • All Connectors - Added support in DEVICE_CONTROL type of events to the integration's connectors.
  • Exchange
    • Exchange
      • Exchange Mail Connector v2, Exchange Mail Connector v2 with Oauth Authentication - New parameters "Original Received Mail Prefix" and "Attached Mail File Prefix" added, in order to prepend a prefix to the keys (to,from,subject...) extracted from the original email received in the monitored mailbox, and attached mail files respectively.
      • Exchange Mail Connector v2, Exchange Mail Connector v2 with Oauth Authentication - Added support for different event types, in case of ingesting an original EML received in the monitored mailbox, and when ingesting as an attached EML. Please refer to our documentation for additional information
  • Mimecast
    • Mimecast
      • NEW! New integration "Mimecast"

Actions

  • Exchange
    • Extract EML Data
      • Extract EML Data - Improved functionality of regex matching within the action.
  • Office365CloudAppSecurity
    • Close Alert
      • NEW! Added new action "Close Alert". Actions "Bulk Resolve Alert" and "Dismiss Alert" are deprecated due to API changes from Microsoft. Make sure to replace those action with "Close Alert" action in your playbooks.
  • Siemplify
    • Create Entity
      • REGRESSIVE! Create Entity Action - added "Delimiter" parameter, allowing to specify a custom delimiter to be used when creating entities via this action. Note - For Siemplify platform versions 5.6.0 inclusive and 5.6.2 exclusive - please make sure to read our usage guide here: https://integrations.siemplify.co/doc/siemplify#create-entity

Connectors

  • CBCloud
    • VMware Carbon Black Cloud Alerts and Events Baseline Connector
      • VMware Carbon Black Cloud Alerts and Events Baseline Connector - Improved the backlog mechanism logic and functionality.
      • VMware Carbon Black Cloud Baseline Connector - "Max Backlog Alerts per Cycle" parameter added.
    • VMware Carbon Black Cloud Alerts and Events Tracking Connector
      • VMware Carbon Black Cloud Alerts and Events Tracking Connector - Added a backlog logic to the connector, to better track alerts and events.
      • VMware Carbon Black Cloud Alerts and Events Tracking Connector - "Max Backlog Alerts per Cycle" parameter added
  • RSANetWitnessPlatform
    • RSA NetWitness Platform - Incidents Connector
      • Improved "Incidents Connector". Added support for configuring broker and concentrator, from which additional information is ingested into Siemplify.
