Siemplify Marketplace Release Notes - 14.5

Published on 23 February, 2021

Integrations

  • Anomali
    • AlienVaultTI
      • Added integration support for the Playbook Simulator feature, allowing you to build, test and edit your workflow logic in a pre-production environment.
    • Anomali
      • Fixed a bug, where integration will treat invalid API Root as valid.
  • CBCloud
    • CBCloud
      • Fixed connector alerts pagination logic for all CB connectors.
      • Updated VMware Carbon Black Cloud Alerts and Events Tracking Connector alert ingestion logic.
      • Updated whitelist handling logic
      • New "Watchlist Name Filter" parameter added in both Events Baseline and Events Tracking connector to filter out only specific alerts with the type “Watchlist“ to be ingested based on the watchlist name
  • ConnectWise
    • ConnectWise
      • Updated Create Ticket and Create Alert Ticket actions descriptions.
  • CrowdStrikeFalcon
    • Crowd Strike Falcon
      • Added ability to provide API root in the "Events Streaming Connector" and integration configuration
  • F5BIGIPAccessPolicyManager
    • F5BIGIPAccessPolicyManager
      • NEW! New BIG-IP Access Policy Manager integration added.

Actions

  • Exchange
    • Download Attachments
      • Fixed decoding issue in the Attachment File's processing to handle more available use cases.
  • LogPoint
    • Update Incident Status
      • NEW! Added new action "Update Incident Status".
  • SSH
    • Run Command
      • Updated JSON result, so that it can be used easier in the playbooks.
  • ServiceNow
    • Ping
      • Fixed a bug, where action "Ping" will take too long to execute.

Jobs

  • RSAArcher
    • Sync Security Incidents
      • NEW! Added new job "Sync Security Incidents".

Connectors

  • CBCloud
    • VMware Carbon Black Cloud Alerts and Events Baseline Connector
      • Updated the connector logic so the connector can subsequently try to ingest alerts it previously failed to process.
  • CrowdStrikeFalcon
    • Crowdstrike Falcon Detections Connector
      • NEW! Added new connector "Detections Connector"
  • LogPoint
    • Incident Connector
      • NEW! Added new connector "Incident Connector".
