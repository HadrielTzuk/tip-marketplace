Siemplify Marketplace 12.17 Release Notes

Published on September 11th,  2019

RELEASE TECHNICAL DETAILS:

  • This marketplace version can only be installed on Siemplify 5.1 and up
  • Integration Changes:
    • Removed all predefined Ontology files from ALL Integration (Feature Got deprecated) (TIPG-40)
    • PaloAlto Cortex XDR (TIPG-38) - New Integration
      • Query Action
      • XDR Connector
    • AlienVault Anywhere (TIPG-36)
      • Added support to new V2 API. User has to choose product version (v1 \ v2) in Integration configuration (Configuration needed)
  • Actions:
    • Exchange
      • SendEmailAndWait 
        • Added support for Exchange 2010, to save email GUID in subject, instead of email ID (as it’s not supported prior to 2013) (TIPG-61)
    • Siemplify
      • RunRemote (publisher-agent execution)
        • Added support for Virtual Enviroments (TIPG-213)
  • Jobs
    • Siemplify
      • Cases Collector
        • Added Response Ack (confirmation back to publisher) on successful cases collection. (TIPG-242)
  • Connectors:
    • Email - Email Connector
      • Improved URL extraction Regex’s (TIPG-51, TIPG-55)
    • Email - Email EML Connector
      • Improved URL extraction Regex’s (TIPG-51, TIPG-55)
    • Email - Imap EML Connector
      • Improved URL extraction Regex’s (TIPG-51, TIPG-55)
    • Exchange - EML Connector
      • Added support to outlook .msg files (TIPG-160)
      • Improved URL extraction Regex’s (TIPG-51, TIPG-55)
    • AlienVault Anywhere - Connector (TIPG-153)
      • Added Whitelist Filter params to connector
        • Show Suppressed
        • Rule Intent
        • Rule Strategy
        • Priority
        • Rule Method
