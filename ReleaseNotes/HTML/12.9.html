Siemplify Marketplace Release Notes - 12.9

Published on 14 July, 2020

Integrations

  • ElasticSearch
    • ElasticSearch
      • Added libraries to the integration.
  • ElasticSearchV7
    • ElasticSearchV7
      • Added libraries to the integration.
  • IronPort
    • IronPort
      • Integration configuration page updated: - added separate fields to specify AsyncOS REST API and SSH ports - added a "Use SSL" checkbox to specify whether AsynOS REST API should be accessed over HTTP or HTTPS.
  • McAfeeESM
    • McAfee ESM
      • Added support for McAfee ESM version 11.2 and 11.3.
  • MicrosoftTeams
    • MicrosoftTeams
      • NEW! Added support for different redirect URLs for configuration.
  • RecordedFuture
    • RecordedFuture
      • Added ability to get information about related entities from enrichment actions.
      • Updated output message for enrichment actions, when no entities were found.
  • ServiceNow
    • ServiceNow
      • Added ability to change the default table that will be used in incidents actions.
      • Minor improvements across the whole integration.

Actions

  • AzureActiveDirectory
    • List Groups
      • Removed the "Order By Field" parameter due to API limitations.
      • Added support for pagination and added a non mandatory "Results Limit" parameter to limit the number of groups that are returned by the action.
    • List Users
      • Updated the possible values of the "Order By Field" parameter due to API limitations.
      • Added "Mobile Phone" column to the user data table.
      • Added support for pagination and added a non mandatory "Results Limit" parameter to limit the number of users that are returned by the action.
      • Updated the JSON results example to match the actual returned data.
  • CSV
    • CSV Search By String
      • Fixed a bug that caused incorrect row count to be returned by the action.
  • IronPort
    • GetAllRecipientsBySubject
      • Action refactored. Action now supports unicode in email subjects. Action input parameters were updated: - New mandatory action input parameter was added - Search Emails for Last X Days. Parameter has a default value. - New mandatory action input parameter was added - Max Recipients to Return. Parameter has a default value.
    • GetAllRecipientsBySender
      • Action refactored. Action input parameters were updated: - New mandatory action input parameter was added - Search Emails for Last X Days. Parameter has a default value. - New mandatory action input parameter was added - Max Recipients to Return. Parameter has a default value.
    • Ping
      • Action refactored.
    • AddSendertoBlocklsit
      • Action refactored.
    • GetReport
      • Action refactored. In action, mail_sender_ip_hostname_detail and mail_incoming_ip_hostname_detail reports work based on Siemplify IP or Host entities; mail_users_detail works on Siemplify User entity (with email address). Other reports are working without Siemplify entities. Action input parameters were updated: - New mandatory action input parameter was added - Search Reports Data for Last X Days. Parameter has a default value. - New mandatory action input parameter was added - Max Records to Return. Parameter has a default value.
  • MISP
    • Get Related Events
      • Added "Events Limit" parameter to limit the number of events fetched by the action.
    • Add Tag To An Attribute
      • Added ability to directly specify in which object of the event to add tag to attribute.
    • Remove Tag From An Attribute
      • Added ability to directly specify in which object of the event to remove attribute.
    • Delete An Attribute
      • Added ability to directly specify in which object of the event to add tag to attribute.
  • McAfeeEPO
    • Get System Information
      • Added support for tasks that contain Unicode data in their name.
  • MicrosoftTeams
    • Generate Token
      • Updated error message when URL without code parameter is used for authorization.
    • Wait For Reply
      • NEW! Added new action Wait For Reply.
    • List Teams
      • Fixed List Teams to work with unicode
    • List Users
      • Fixed List Users to work with unicode
  • SentinelOneV2
    • Create Path Exclusion Record
      • NEW! Added new properties ‘Site IDs’ and ‘Group IDs’ in the action and align action results.
    • Create Hash Exclusion Record
      • NEW! NEW action for creating a hash exclusion for a white list.
    • Mitigate Threat
      • NEW! NEW action for applying a mitigation action to a group of threats.
    • Mark As Threat
      • NEW! NEW action for marking suspicious threats as a threat.
    • Resolve Threat
      • NEW! NEW action for resolving threats.
    • Get Threats
      • NEW! NEW action for getting threats list.
  • ServiceNow
    • CreateRecord
      • Added JSON Result to action "Create Record".
  • Slack
    • WaitForReply
      • Fixed a bug, where action wouldn't execute even with proper parameters.
