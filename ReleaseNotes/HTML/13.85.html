Siemplify Marketplace Release Notes - 13.85

Published on 24 November, 2020

Integrations

  • AWSGuardDuty
    • AWSGuardDuty
      • New Actions: Create Sample Findings, Archive/Unarchive Findings, Update Findings Feedback, Get Detector Details, Delete a Trusted IP List/Threat Intelligence Set, Create/Update/Delete a detector, List Detectors
  • Exchange
    • Exchange
      • Exchange - Added integration parameter - "Autodiscover" - To enable the Autodiscover service using the integration with Exchange. You can read more on our documentation site.
      • Exchange - Fixed a bug where async actions failed due to Exchange network errors. Added a fallback mechanism to handle the issue. Also, added new integration parameter to handle the issues better - Auto Discover. You can read more about it in the documentation site.
