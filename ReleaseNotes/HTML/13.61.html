Siemplify Marketplace Release Notes - 13.61

Published on 21 October, 2020

Actions

  • Exchange
    • Wait for mail from user
      • NEW! New "Wait for Mail from User" action added.
    • Search Mails
      • Search Mails action updated.
    • Delete Mail
      • Delete Mail action updated.
      • Exchange: Fixed a bug, where "Delete from All Mailboxes" = true and "Delete All Matching Emails" = false caused an unexpected behavior.
    • Move Mail To Folder
      • Move Mail to Folder action updated.
    • Save Mail Attachments To The Case
      • NEW! New "Save Mail Attachments To The Case" action added.
    • Download Attachments
      • Download Attachments action updated. Fix: "Download Attachments" action now supports downloading any type of attachment.
    • Get Account Out Of Facility Settings
      • NEW! Added new action "Get Account Out Of Facility Settings"

Connectors

  • Exchange
    • Exchange Mail Connector v2
      • NEW! Exchange Mail Connector V2 added. Connector V2 can ingest both eml, msg or ics files received with email to the monitored mailbox, and "regular" emails. Other enhancements were also made comparing to Exchange and EML connectors.
