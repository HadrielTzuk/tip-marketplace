Siemplify Marketplace Release Notes - 12.14

Published on 07 July 2019

Actions

  • Logrythm
      • Fixed missing dependencies for the integration
  • Intsights
      • Fixed incorrect integration description
  • MxToolBox
      • Fixed test connectivity mechanism
  • Email
    • Added sender display name parameter to integration configuration
  • Exchange
    • Send Email
      • Added CC, BCC support
    • Send email and wait
      • Added CC, BCC support
      • Added checkbox that allows placing the attachments of a response email on the case wall as evidence.
    • Send Mail HTML
      • Added CC, BCC support
  • VirusTotal
    • Prevent action failure when an entity doesn’t exist on VirusTotal 
  • McafeeEPO
    • Get System Information
      • Updated action behavior to search for exact IP / HOSTNAME instead of "contains"
  • CSV
    • CSV Enrichment By Entity (New Action)
      • Search for entity records in CSV files (in a specific folder) and enrich entities accordingly
    • CSV Enrichment By Filter (New Action)
      • Check for string records in CSV files (in a specific folder)
  • Endgame
    • - List all endpoints
    • - Searches for running files
    • - Searches for network connections
    • - Searches for running processes
    • - Searches for a registry key or value name
    • HuntUser (New Action) - Searches the network for logged in users
  • Siemplify
    • Create Or Update Entity Properties
      • Added Unicode support

Connectors

  • Splunk
    • Changed connector’s environment field to non-mandatory (leaving field empty will use the default environment drop-down menu)
  • Mcafee - ESM Connector
    • Fixed missing dependencies for the integration
  • Arcsight - Correlations Connector
    • Added Correlations name Blacklist config file. Can be found under C:\Siemplify_Server\Scripting\SiemplifyConnectorExecution\<Connector run folder>\blacklist.json
  • Endgame - Connector
    • Fetches alerts based on running timestamps and various filters
  • Email integration, Exchange integration
    • Email Connector, Email EML Connector
      • In case of forwarded mail from Outlook - added support for parsing the metadata of the original sender (and not the forwarded mailbox)
      • Added the following fields to the email event:
        •  Latest mail forwarder
        • Latest mail recipient
