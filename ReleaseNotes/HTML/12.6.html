Actions

  • APIVoid
    • Get Ip Reputation
      • Updated action logic to skip internal IPs.
      • Updated action logic to prevent it from failing because of a single entity failure. Action will be considered as successful if at least one entity is enriched.
  • CiscoAMP
    • Get Computers By Network Activity (Ip)
      • Fixed a bug that caused the action to always output a success message, even if it didn't find any information.
  • CiscoUmbrella
    • Get Whois
      • Fixed a parsing issue that caused the action to fail.
  • SymantecATP
    • Add Comment To Incident
      • Fixed a bug, where action wouldn't work even with valid action parameters.
    • Enrich File Hash
      • Fixed a bug, where action wouldn't work even with valid hashes.
    • Get Events Free Query
      • Fixed a bug, where action wouldn't work with valid queries.
    • Close Incident Action
      • Fixed a bug, where action wouldn't work with valid action parameters.
    • Get Events Free Entity
      • Fixed a bug, where action wouldn't work even with valid action parameters.
    • Add To WhiteList
      • Added a special message that Symantec ATP doesn't support MD5 hashes in whitelist.
    • Isolate Endpoint
      • Fixed a bug, where action wouldn't find the existing host.
    • Rejoin Endpoint
      • Fixed a bug, where action wouldn't find the existing host.
    • Update Incident Resolution
      • NEW! New action Update Incident Resolution
    • Delete BlackList Policy
      • NEW! New action Delete BlackList Policy
    • Get Incident Comments
      • NEW! New action Get Incident Comments
  • VirusTotal
    • Get Domain Report
      • Fixed a bug that caused the action to fail on entities that don't have categories in the VirusTotal report.
  • XForce
    • Get IP Malware
      • Fixed a bug in CSV table building that caused the action to fail on non-malicious IPs.
