Siemplify Marketplace Release Notes - 15.3

Published on 15 June, 2021

Integrations

  • QualysEDR
    • QualysEDR
      • NEW! New integration "Qualys EDR".

Actions

  • CiscoOrbital
    • Execute Query
      • Fixed a bug, where action "Execute Query" was not working correctly, due to Cisco Orbital's API changes.
  • MISP
    • Add Attribute
      • Added support for additional categories in action "Add Attribute".
      • Changed "Add Attribute" action to use the defaults provided by MISP API, instead of using "External Analysis" by default.
      • "Add Attribute" action - Minor bug fixes in the "For Intrusion Detection System" and "Distribution" parameters.
  • RecordedFuture
    • Enrich IOC
      • Enrich IOC action now supports enriching domains with Recorded Future, when using the URL entity type in Siemplify
  • SentinelOneV2
    • Update Analyst Verdict
      • Fixed a bug, where action "Update Analyst Verdict" wouldn't update the analyst verdict.
  • Siemplify
    • Update Case Description
      • Added a new action "Update Case Description".
  • VMRay
    • Upload File And Get Report
      • Fixed a bug, where action "Upload File And Get Report" was not working correctly, due to VMRay's API changes.

Connectors

  • Sophos
    • Sophos Central - Alerts Connector
      • NEW! Added new connector "Alerts Connector".
  • Splunk
    • Notable Events Connector
      • Added an ability to get notable event data along side base events in "Notable Events Connector" .
      • Improved the "Notable Events Connector" functionality, to properly handle alerts that are missing "Alert Name".
      • Fixed a bug, where "Notable Events Connector" will try to create base events, even if the "Extract Base Events" parameter is disabled. Note: In case you would like to ingest base events, please make sure to enable the "Extract Base Events" parameter.
