Siemplify Marketplace Release Notes - 12.31

Published on 28 April, 2020

Integrations

  • AutoFocus
    • AutoFocus
      • Added support for Unicode data
  • Cylance
    • Cylance
      • Removed sensitive data from integration code.
  • VirusTotal
    • VirusTotal
      • Updated error handling for requests that exceeded their rate limit and for forbidden requests in cases of using wrong API key or making a request to a Private API without having the appropriate privileges.

Actions

  • ElasticSearch
    • AdvancesESSearch
      • Updated time-based filtering logic and added a new non-mandatory parameter - "Timestamp Field" for specifying the name of the field to run time-based filtering against.
    • SimpleESSearch
      • Improved the search logic that uses Lucene syntax.
  • EmailV2
    • WaitForEmailFromUser
      • REGRESSIVE! Updated action's JSON results for better usability in playbooks.
  • Office365CloudAppSecurity
    • Get User Related Activities
      • Adjusted the action as the original Users can contain lower and uppercase letters
  • UrlScanIo
    • Ping
      • Updated ping action to use www.siemplify.co for test scan instead of blacklisted www.google.com.
  • VirusTotal
    • ScanURL
      • Added error handling to comments fetching.
