Siemplify Marketplace Release Notes - 14.95

Published on 27 April, 2021

Integrations

  • RSAArcher
    • RSAArcher
      • Fixed a bug, where action "Update Incident" and "Create Incident" were not working with checkbox parameters.
      • Added an ability to work with mapping files via URLs in actions "Create Incident" and "Update Incident".
  • TruSTAR
    • TruSTAR
      • NEW! New integration "TruSTAR".

Actions

  • MISP
    • Enrich Entities
      • Fixed a bug, where parameter "Number of attributes to return" is not having an impact on how many attributes are returned. Removed enrichment fields based on the entity related attributes.
  • PaloAltoPanorama
    • Block URLs, Unblock URLs
      • Improved processing of the URLs that contained special characters in actions "Unblock URLs" and "Block URLs"
  • SiemplifyThreatFuse
    • Get Related Associations
      • Added statistics information regarding IOCs related associations in action "Get Related Associations".

Connectors

  • LogRhythm
    • LogRhythm - Rest API Alarms Connector
      • NEW! New connector "Rest API Alarms Connector".
  • Office365CloudAppSecurity
    • Office 365 CloudApp Security Connector
      • Office 365 CloudApp Security Connector alert fetching logic is updated to minimize the chance of missing alerts for ingestion. New connector parameter "Alerts Padding Period" should be adjusted accordingly to the environment to avoid losing alerts for ingestion.
  • QRadar
    • QRadar Correlation Events Connector V2
      • Improved logging in the "QRadar Correlation Events Connector V2" connector
