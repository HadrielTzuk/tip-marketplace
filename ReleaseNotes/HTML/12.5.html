Siemplify Marketplace Release Notes - 12.5

Published on 19 May, 2020

Integrations

  • AutoFocus
    • AutoFocus
      • Results coming from all Autofocus "hunt" actions will be returned in a JSON format.
      • Fixed a bug in all Autofocus "hunt" actions that prevented proper display of tables.
      • Applied "Results Limit" to table rows in all Autofocus "hunt" actions.
  • CSV
    • CSV
      • Added support for unicode characters in output table.
  • McAfeeEPO
    • McAfeeEPO
      • Improved logging for all of the actions.
  • PaloAltoNGFW
    • PaloAltoNGFW
      • Removed sensitive data from logs
  • SCCM
    • SCCM
      • Updated dependencies to prevent connection errors on Linux machines.
  • ServiceNow
    • ServiceNow
      • Removed sensitive data from logs.
  • Splunk
    • Splunk
      • Prevented actions from revealing sensitive data in the logs.
