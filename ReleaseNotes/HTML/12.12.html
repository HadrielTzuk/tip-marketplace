Siemplify Marketplace Release Notes - 12.12

Published on 09 June 2019

Actions

  • Siemplify
    • Get Similar Cases
      • Minor bug fixes
  • APIvoid - New Integration (formerly - IPvoid + URLvoid)
    • Get Domain Reputation - New Action
    • Get IP Reputation - New Action
  • URLVoid
    • Added “verify SSL” checkbox in the integration configuration
    • Integration and configuration updated to support new API
  • IPVoid
    • Added “verify SSL” checkbox in the integration configuration
    • Integration and configuration updated to support new API
  • VmRay
    • Test connectivity - minor bug fix
    • Added “verify SSL” checkbox in the integration configuration
  • Zabbix
    • Execute Script - New Action
      • Allow the user to run scripts on hosts via Zabbix
  • Email
    • Send email
      • Added CC Param
    • Send email and wait
      • Added CC Param
      • Removed UUID from subject (No longer needed)
      • Minor bug fix in fetching emails after sending - added support for special characters in action
  • Exchange
    • Send email
      • Added CC Param
    • Send Mail HTML
      • Added CC Param
    • Send email and wait
      • Removed UUID from subject (No longer needed)
      • Added CC Param
  • CBDefense
    • Integration updated to use CB’s new SDK (cbapi-1.4.4)
  • CBProtection
    • Integration updated to use CB’s new SDK (cbapi-1.4.4)
  • CBResponse
    • Integration updated to use CB’s new SDK (cbapi-1.4.4)
  • ThreatConnect
    • Support ThreatConnect’s rebranding
  • Siemplify Publisher - Deprecated
    • Publisher integration was removed from the marketplace as Siemplify 5.0 architecture no longer needs this integration
  • Expression Builder - Json Example Updated
    • CiscoAMP
    • CiscoThreatGrid
    • McAfeeESM
    • ReversinglabsTitanium

Connectors

  • ArcSight - Correlations connector
    • Sensitive data was removed from local logs
  • Email Connector
    • Added proxy support in the connector configuration
    • Improved Forwarded email detection
  • Email EML connector
    • Added proxy support in the connector configuration
  • Exchange EML Connector - New Connector
    • New Exchange connector that ingests the contents of an attached EML into a case as new alert (as opposed to the main email)
  • Exchange Mail Connector
    • Extended support for uncommon EML formats
    • Added proxy support in the connector configuration
  • McAfeeESM Correlations Connector
    • Added correlation’s event fields into alert
  • Mcafee ESM Connector (Alarms)
    • Improved environment ingestion logic
      • Added 'Use Mcafee Environments' checkbox - ingest cases to Siemplify environment by Mcafee multi-tenant mechanism
      • Added map.json file to support the use case of unequal environment names in Mcafee and Siemplify
    • Added option to exclude alerts from Siemplify connector using a blacklist
  • QRadar New Correlations Events Connector
    • Improved environment ingestion logic
      • Added 'Use Qradar Environments' checkbox - ingest cases to Siemplify environment by Qradar multi-tenant mechanism
      • Added map.json file to support the use case of unequal environment names in Qradar and Siemplify
