Siemplify Marketplace Release Notes - 12.4

Published on 07 May, 2020

Integrations

  • Splunk
    • Splunk Pull Connector
      • NEW! Added Splunk Pull Connector. Customers who are going to update the marketplace using zip, please follow these steps in order to successfully update the Splunk integration: 1. Update the marketplace using the provided zip file. 2. Connect to your server using or RDP (for 5.1) or ssh (for 5.3) and go to the following path: On 5.1: C:\Siemplify_Server\MarketPlace\Integrations\Splunk\Dependencies\cross. On 5.3: /opt/siemplify/siemplify_server/Marketplace/Integrations/Splunk/Dependencies/cross. And make sure you only have the latest dependencies in this folder, you can either compare this folder content with the latest integration in the marketplace zip or leave only the newest dependencies on your server, e.g if you have TIPCommon-1.0.9-py2.. and TIPCommon-1.0.10-pyâ€¦ remove the TIPCommon-1.0.9-py2.. from your server. Then update the integration to the newest version.

Actions

  • ActiveDirectory
    • Enrich Entities
      • Fixed LDAP filters in Enrich Entities action
  • MISP
    • Enrich entities
      • Fixed the bug causing high CPU usage
  • MitreAttck
    • Get Technique Details
      • Added support for the External ID of the attack technique.
    • Get Mitigations
      • Added support for the different attack technique identifiers (Name, Attack ID, External ID). After the upgrade, there is a need to update all Playbooks that use this action. 1. Go to the playbook 2. Find the action 3. Configure new mandatory parameter Identifier Type.
    • Get Associated Intrusions
      • Added support for the different attack technique identifiers (Name, Attack ID, External ID). After the upgrade, there is a need to update all Playbooks that use this action.
  • SentinelOneV2
    • Get Hash Reputation
      • Added new ability to get reputation details for hashes.
  • VirusTotal
    • Scan URL
      • Fixed issues for URLs that contain both upper and lower case letters
