Siemplify Marketplace Release Notes - 13.0

Published on 28 July, 2020

Integrations

  • AlienVaultAnywhere
    • AlienVault USM Anywhere
      • Migrated integration from Python2 to Python3. Newer version is only supported on Siemplify 5.3 and higher.
  • MicrosoftGraphSecurity
    • Microsoft Graph Security
      • Refactored integration to support Python 3.7
  • PaloAltoPanorama
    • Palo Alto Panorama
      • Removed sensitive data from API requests and logs.
      • Minor improvements across documentation and integration.

Actions

  • APIVoid
    • Get URL Reputation
      • NEW! New action for getting URL reputation.
    • Get Screenshot
      • NEW! New action for capturing screenshots of URLs.
    • Verify Email
      • NEW! New action for verifying an email.
  • MicrosoftGraphSecurity
    • List Alerts
      • Replaced the JSON visual representation of alerts with a CSV table
    • Get Alert
      • Updated the alert details table to be more readable and user friendly
  • PaloAltoPanorama
    • Search Logs
      • NEW! Added new action "Search Logs".
    • Get Correlated Traffic Between IPs
      • NEW! Added new action "Get Correlated Traffic Between IPs".
  • Siemplify
    • Create Entity
      • Mark new entity with enrichment property - is_new_entity

Jobs

  • Jira
    • Sync Closure
      • NEW! Sync Closure job added.

Connectors

  • ElasticSearch
    • ElasticSearch Connector
      • Fixed a bug that prevented the connector from updating the timestamp if all the cases are considered as overflow.
  • EmailV2
    • Generic IMAP Email Connector
      • Generic IMAP Connector parameters update.
      • Updated regex for Subject in Generic IMAP Email Connector.
      • Updated regex for URLs in Generic IMAP Email Connector.
      • Fixed extraction of PNG and PDF files from emails sent from Built-in Mac OS email clients.
      • Added extraction of HTML inline images in Generic IMAP Email Connector.
  • McAfeeESM
    • McAfee ESM Connector
      • Fixed a bug in the caching mechanism to prevent duplicate alarm processing.
  • PaloAltoPanorama
    • Threat Log Connector
      • NEW! Added ability to ingest threat logs into Siemplify.
