Siemplify Marketplace Release Notes - 13.2

Published on 25 August, 2020

Integrations

  • Arcsight
    • Arcsight
      • API requests used in the integration were updated to be more secure.
  • ThreatQ
    • ThreatQ
      • Updated integration to work with Signature and Task object types properly.

Actions

  • AlienVaultAnywhere
    • List Events
      • NEW! New action for searching events in Alientvault (Product Version V2)
    • Get Alarm Details
      • NEW! New action for getting alarm full details (Product Version V2)
  • Arcsight
    • Search
      • NEW! Add limitation for how many results should be available in the response to the action "Search".
  • CSV
    • CSV Search By String
      • Add "File Encoding Types" parameter to allow user to specify the expected encoding of the CSV files.
    • CSV Search By Entity
      • Add "File Encoding Types" parameter to allow user to specify the expected encoding of the CSV files.
  • ConnectWise
    • Update Ticket
      • Added new parameters to the Update Ticket action
    • Create Ticket
      • Added an option to assign ticket to user
    • Create Alerts Ticket
      • Added an option to assign ticket to user
  • Splunk
    • Update Notable Events
      • NEW! New action "Update Notable Events".
  • ThreatQ
    • Create Event
      • NEW! Added new action Create Event.
    • Update Indicator Status
      • NEW! Added new action Update Indicator Status.
    • Update Indicator Score
      • NEW! Added new action Update Indicator Score.
    • Link Entities
      • NEW! Added new action Link Entities.
    • Create Indicator
      • NEW! Added new "Indicator Type" in action "Create Indicator", which automatically identifies hashes.
    • Link Entities To Object
      • NEW! Added new action "Link Entities To Object".
    • List Events
      • NEW! Added new action "List Events".
  • UrlScanIo
    • Ping
      • Updated ping action to use not blacklisted URL for test scan instead of blacklisted www.siemplify.co.
  • VirusTotal
    • Upload And Scan Files
      • Fix a bug that causes the action to fail on all executions.
    • Scan Hash
      • Added dedicated error messages for forbidden API requests (due to usage of public API key).
  • Zscaler
    • Get Url Categories
      • Add a new "Display URL" parameter, to determine whether to display categories' URLs or not via CSV table.
    • Lookup Entity
      • Add the ability to send multiple urls per single API request.
