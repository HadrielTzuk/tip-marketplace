Siemplify Marketplace Release Notes - 14.75

Published on 30 March, 2021

Integrations

  • MicrosoftAzureSentinel
    • Microsoft Azure Sentinel
      • Added OAUTH2 Login Endpoint Url parameter in Azure Sentinel Connectors and Integration.
  • TrendMicroApexCentral
    • TrendMicroApexCentral
      • NEW! New integration "Trend Micro Apex Central"

Actions

  • ServiceNow
    • Add Attachment
      • Fixed a bug, where action "Add Attachment" wasn't working, because incorrect content-type header was applied.
    • Get Incidents
      • Updated JSON example for action "Get Incidents"

Connectors

  • CBCloud
    • VMware Carbon Black Cloud Alerts and Events Baseline Connector
      • Added a fallback mechanism in the Events Baseline Connector, to better handle cases of bad API responses and temporary missing info cases.
  • MicrosoftAzureSentinel
    • Microsoft Azure Sentinel Incident Connector v2
      • Added fallback logic in Microsoft Azure Sentinel Incident Connector v2 for Vendor Field Name, Product Field Name and Event Field Name fields. After the integration and connector upgrade, its recommended to change the value of Product Field Name and Event Field Name to product_type and event_type respectively for this new functionality to work.
      • Enhanced the Microsoft Azure Sentinel Incident Connector v2 logic for handling Start Time and End Time fields, connector will use a pre-defined list of fallback values for Start Time and End Time fields.
      • Updated Microsoft Azure Sentinel Connector v2 backlog management logic.
      • Fallback fields debug parameter added in Azure Sentinel Connector v2.
